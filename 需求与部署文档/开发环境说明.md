# 开发环境说明

## 一键启动

```bash
./start-dev.sh
```

按 `Ctrl+C` 停止所有服务。

## 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| Server API | http://localhost:8000 | 后端接口 |
| API 文档 | http://localhost:8000/api/docs | Swagger 文档 |
| PC 用户端 | http://localhost:3001 | 桌面端用户界面 |
| Mobile 移动端 | http://localhost:3002 | 移动端 H5 |
| Admin 管理后台 | http://localhost:3003 | 运营管理后台 |

## 单独启动

如需单独启动某个服务：

```bash
# 后端
cd packages/server && pnpm dev

# PC 用户端
cd packages/pc && pnpm dev

# 移动端
cd packages/mobile && pnpm dev

# 管理后台
cd packages/admin && pnpm dev
```

## 数据库

```bash
# 进入 server 目录
cd packages/server

# 生成 Prisma Client
pnpm db:generate

# 同步数据库结构
pnpm db:push

# 填充种子数据
pnpm db:seed

# 打开数据库管理界面
pnpm db:studio
```
